# 단일 엔드포인트에서 여러 최신 SOTA LLM을 Scale-to-Zero로 배포하기

이 노트북에서는 오토스케일링과 스케일-투-제로(scale-to-zero) 기능을 통해 비용을 최적화하면서 단일 SageMaker 엔드포인트에 여러 최첨단 파운데이션 모델을 배포하는 방법을 배웁니다.

먼저 Fast Model Loader를 사용한 모델 최적화부터 시작한 후 오토스케일링 정책을 구성하고, 마지막으로 트래픽 테스트를 통해 스케일링 동작을 확인합니다.

### 1. Fast Model Loader를 사용한 모델 최적화

이 섹션에서는 다음을 수행합니다:
-  SageMaker ModelBuilder를 사용하여 **Llama 3.1 8B Instruct** 설정
-  더 빠른 배포를 위해 모델 가중치를 S3에서 GPU로 스트리밍하는 **Fast Model Loader** 구성
-  **Inference Component 기반 엔드포인트**에 모델 배포

### 2. Scale-to-Zero를 포함한 오토스케일링 구성

이 섹션에서는 오토스케일링을 구현합니다:
-  1→N 스케일링을 위한 **타깃 트래킹(Target Tracking) 정책** 설정
-  0→1 스케일링을 위한 **스텝 스케일링(Step Scaling) 정책** 구성

### 3. 라이브 트래픽으로 스케일링 동작 테스트

이 섹션에서는 스케일링 설정을 검증합니다:
-  트래픽이 없을 때의 **스케일-투-제로 동작** 확인
-  오토스케일링을 유발하기 위한 **동시 요청 부하** 생성
-  실시간으로 **인퍼런스 컴포넌트 스케일링** 모니터링

### 4. (선택) 동일 엔드포인트에 여러 모델 배포

이 선택적 섹션에서는 다음을 수행합니다:
-  기존 엔드포인트에 두 번째 모델로 **Mistral 7B Instruct** 배포
-  각 모델에 대해 **독립적인 오토스케일링** 구성
